Построение линейной модели для оценки тональности отзыва на фильмы IMDB (создание из текста набора признаков, который можно будет обработать линейной моделью). 
Есть данные целевой группы и известная тональность отзывов из целевой группы (1 - положительный, 0 - отрицательный), а также данные контрольной группы.

- Поработаем с текстом отзывов (уберем мусор типа лишних пробелов, тегов, всех небуквенных символов, приведем все слова к нижнему регистру).
- Уберем стоп-слова (from nltk.corpus import stopwords)
- Оставшиеся слова приведем к нормальной форме (лемматизация и стемминг - from nltk.stem.porter import PorterStemmer, from nltk.stem import LancasterStemmer, from nltk.stem import WordNetLemmatizer)
- Сделаем векторизацию (HashingVectorizer, Tf-idf.Vectorizer)
- Обучим LogisticRegression() на тестовой группе и с помощью полученных результатов сделаем прогноз для контрольной группы
- Проверим качество модели на кросс-валидации
- Оценим переобучение
- Влияет ли как-то регуляризация на результат? Логрегрессия поддерживает l1 и l2 регуляризации.
